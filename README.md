Mag7 Market & Sentiment Intelligence Platform
A fully automated market-intelligence pipeline that extracts live stock data, sentiment signals, and market benchmarks for the Magnificent 7 (MAG7), compares them against the S&P 500, and serves interactive insights through a Streamlit dashboard.
Built with:
Meltano – ELT extraction & loading
BigQuery – Cloud warehouse
Dagster – Orchestration, schedules, sensors
Streamlit – Interactive UI
Docker – Reproducible environment
The entire pipeline auto-runs every 00:00 SGT to keep data fresh.
